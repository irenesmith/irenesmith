What is image slicing? It means dividing an image into smaller images. Why would you want to do that? For one thing, it is one way to make an image that links to several different places.

As an example of image slicing, the above image was sliced into nine equal pieces and the center image was linked to the first page of my portfolio. Go ahead and click it if you want to. The link will open in a separate window so you won't lose your place.

The display on this updated version uses CSS Grid (yes, again!) to keep the individual images lined up. If you'd like to see the grid layout, use the Page Inspector tool to access the Grid Inspector or take a look at my screenshot of this page to see a static version of the grid.
