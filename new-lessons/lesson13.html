Lesson 13 is all about mouseovers. What is a mouseover? A mouseover is when the graphics on a page change when the mouse passes over them. It does this using JavaScript code that handles the mouseover and mouseout events. For more information about responding to mouse events in JavaScript, take a look at the list of Mouse Events on MDN Web docs.

If you want to see an example, try moving the mouse over the title graphic and watch what happens. Using mouseovers does more than add a little animation to your page too. Mouseovers can be used to provide feedback or more information to visitors.

You'll notice that the buttons on this page also change when you point at them with the mouse. In all cases, the mouseover effect on this page was created by using two images, one for the regular state of the graphic and the other for when the mouse pointer is over it. JavaScript code takes care of swapping the images when the mouse pointer enters or leaves the screen space over the graphic.
