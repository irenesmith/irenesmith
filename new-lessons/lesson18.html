After providing interesting content, creating good consistent navigation is probably the most important part of web design. After all, if people can't find your way around the site, they'll quickly find their way elsewhere.

There are a number of ways to provide good navigation for a site, including buttons, text links, and menus. This lesson is all about using JavaScript menus to navigate from page to page. If you look at the left-hand side of this page, you'll see a bar that says, "Site Menu." If you hover your mouse pointer over the bar, a menu will slide into view that gives you access to my entire portfolio and my personal website as well. The pages on my personal site will open in a separate window, so feel free to take a look around.

NOTE: I find it interested that, in order to validate the CSS code on this page, I had to change the contents of the title tag and the alt tag for the title graphic to use the word "and" written out rather than the ampersand character "&" and it doesn't make a difference if I use "&amp;" instead because "&amp;" is replaced by a simple & anyway and the CSS validator can't handle it.
