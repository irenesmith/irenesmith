The bottom line in 2019 is don't use tables to layout your pages. The only time you should use a table is when you want to display rows and columns of data. For example, a list of movies with columns that describe each movie, such as Title, Director, Studio, and Description.

The original version of this page talked about how HTML doesn't give you complete control over the layout of a page. When I created this page, the main way to get around that problem was the use of tables.

The original version of this page used tables to arrange the content. This version of the page uses the CSS Grid instead. You can see how I laid out the content if you are using Firefox. Firefox has a grid inspector which you can access by opening the Page Inspector. Select the system menu and, select Web Developer, and then Inspector. Or do it the easy way by hitting Ctrl + Shift + C (Cmd + Shift + C on macOS).
