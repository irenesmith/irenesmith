Drawing on the Canvas

Color:

The drawing code in this example is based on code written by Jeff Duntemann in his book, Delphi Programming Explorer. It uses jQuery to draw on an HTML5 canvas. Each time you click the DRAW button, a different random drawing will be added to the canvas. If you want to clear and start over again, click the CLEAR button.
