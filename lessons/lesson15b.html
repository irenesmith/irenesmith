This page shows you where the boundaries of the table cells fall. It is exactly like the first page for lesson 15, except that the borders have been turned on.

While using tables like this is considered old-fashioned, it's still the most reliable way to lay out a page. You can accomplish pretty much the same thing by using CSS but it's difficult to be sure that all browsers will show the table the same way.

I created another page that shows a different layout. This one uses multiple tables to accomplish its goals. If you're done, however, you can return to the main page for this lesson.
