One of the most frustrating things about HTML is that it doesn't give you complete control over the layout of a page. One of the ways to get around that problem is the use of tables.

For example, this page uses several tables to get things where they belong. In order to show you how I've made this page look the way it does, I've turned on the borders so you can see where the breaks are.

When you are done looking at this page, you can go back to the main page for this lesson or to the version of this page without borders.
