This page has an image that has been displayed using a Snow Applet, a Java applet that modifies the image to make it look as though there was a snowstorm going on in the picture. Unlike a lot of the other photos I've used to create the pages in this portfolio, this is one that I took myself. This is my son. It was taken in December of 2003 the day after the first snow he was old enough to go out in. We had been living in Seattle, Washington for the past four years and during the time we were there, we only had one snowstorm that left anything on the ground for more than an hour or so and only half an inch or so at that. So this was absolutely amazing to him.
