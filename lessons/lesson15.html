One of the most frustrating things about HTML is that it doesn't give you complete control over the layout of a page. One of the ways to get around that problem is the use of tables.

For example, this page uses several tables to get things where they belong. In order to show you how I've made this page look the way it does, I've created a second page that has the borders of the large table turned on so you can see where the breaks are.

There are actually four pages for this lesson:

  1. Lesson 15 - this page
  2. Lesson 15b - this page with the borders turned on
  3. Lesson 15c - a different, more complicated table layout
  4. Lesson 15d - the same as 15c but with the borders on
