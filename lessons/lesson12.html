This page was set up in much the same way as the page for Lesson 11. However, instead of using the standard table border, I've used CSS styles to define a dotted border around my tables. It give a nice look, don't you think? In addition to the borders, I've used JavaScript (from The JavaScript Source) to add a message to the status bar at the bottom of the screen. Of course, it only works in Internet Explorer. Furthermore, since I've downloaded Internet Explorer 7.0 Beta, I'm noticing that a lot of the IE only goodies like status bar messages have gone away. From what I can tell by experimenting, the text in the status bar appears and disappears almost instantaneously.

I also created a second page almost like this one with a different border. Take a look if you are interested.
